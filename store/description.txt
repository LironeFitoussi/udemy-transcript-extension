Udemy Transcript Copier

Copy any Udemy lesson transcript to your clipboard with a single click — no sidebar required.

HOW IT WORKS
A clipboard icon appears directly in the Udemy video player controls (next to Transcript, Captions, and Settings). Click it and the full transcript is fetched instantly via Udemy's API in the background and copied to your clipboard. The button shows a spinner while loading and restores automatically when done.

No sidebar toggling. No page manipulation. No UI side effects.

FEATURES
• One-click copy — works whether the transcript panel is open or closed
• API-powered — fetches directly from Udemy's caption API, not the DOM
• Loading spinner — visual feedback while fetching, prevents double-clicks
• Three copy formats: Plain text, Markdown, or JSON
• Optional metadata — prepend course title, lesson name, and date
• SPA-aware — button stays injected when navigating between lectures
• Lightweight — no background service worker, no external requests

USE CASES
• Feed transcripts into ChatGPT, Claude, or other AI tools instantly
• Save lesson notes for offline reference
• Generate summaries or study guides from course content
• Export to Notion, Obsidian, or any note-taking app

PRIVACY
This extension operates entirely on your device using your existing Udemy session. No transcript content, usage data, or personal information is ever transmitted anywhere. The only permissions used are clipboard access (to copy the transcript) and storage (to save your format preference).

PERMISSIONS EXPLAINED
• clipboard-write: Required to copy the transcript text to your clipboard
• storage: Required to remember your preferred copy format between sessions

VERSION HISTORY

v1.0.1 (latest)
• Fixed a bug where navigating between lectures without a full page reload (Udemy's SPA routing) caused the copy button to fetch the wrong lesson's transcript
• The extension now listens to URL changes in real time — when you switch to a new lecture, the button resets and re-injects itself into the updated control bar
• Lecture ID is now read directly from the URL (always in sync with navigation) instead of relying on a DOM attribute that could lag behind

v1.0.0 — Initial Release
• Injected a clipboard icon button directly into the Udemy video player control bar
• Fetches the full transcript via Udemy's internal caption API — no sidebar interaction required
• Falls back to reading the transcript DOM if the API is unavailable
• Loading spinner during fetch with disabled state to prevent double-clicks
• Three output formats: Plain text, Markdown (with optional heading), and JSON
• Optional metadata header: prepends course title, lesson name, and timestamp
• MutationObserver re-injects the button if Udemy's UI removes it during navigation
